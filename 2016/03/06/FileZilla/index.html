搭建个人FTP服务器

使用FileZilla Server轻松搭建个人FTP服务器

安装

百度搜索FileZilla Server，下载到本地并安装

配置

打开安装好的FileZilla Server

Server Address改为127.0.0.1;其他的不用动,点击OK按钮
进入到主界面，然后点击Edit下的Users进入到Users界面

点击add按钮，添加一个用户

选中该用户，设置访问密码

然后在”Shared folders”下设置将要设为FTP目录的文件夹和操作权限
点击add按钮，添加一个共享文件夹，当然这里也可以增加多个，加错了也可以删除(右击要删除的文件夹，点击remove)

这里演示的是把D盘加进来了

设置操作权限(这里全选了)

最后点击界面左下角的Ok按钮，加入该用户

测试FTP

使用本机或者局域网其他电脑，Windows徽标键+R打开运行窗口，输入ftp:// ，为你电脑的IP地址，按回车就弹出了验证窗口。或者打开一个文件夹在地址栏直接输入 ftp://* 并回车。然后输入刚才设置的用户和密码，回车，是不是可以正常访问了？
如果本机测试的话IP可以为127.0.0.1

点击确定后,需要你输入用户名和密码，就是刚才设置的

点击登录后,就可以正常访问了

用户的连接信息在软件运行窗口都可以实时看到
